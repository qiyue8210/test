第十章 div 和span

   div 块元素，相当于一个容器方便多个元素统一设置格式；span 内联元素，相当于一个特例，从其他元素中单独设置格式，配合class元素使用可以设置一批相似元素格式
  1. css中元素的显示方法
    • 单一元素 如p;
    • 多个元素同时使用一个规则，中间用逗号隔开 如h1,h2;
    • 某个类 类名之前是句点 如 p.header;
    • 某个id id名之前是# 如p#header;
    • 子孙的方法 从父元素到子元素，中间空格隔开 如p img#cat；
    • 自定义css规则 开头是@
    • 元素状态 如a:link { color: #007e7e; }

  2. 超文本连接元素状态:link未访问；visited已访问；hover悬停；focus焦点；active活动；

    body p a:link { color: #007e7e; }
body p a:visited { color: #333333; }

  3. 层叠 样式表的优先规则:首要规则，作者>读者>浏览器；其次，选择器特定性越大越优先 计算法则:包含id加100，包含class或伪类加10，包含元素加1。比数大小。最后，在同一个css中看位置，越靠下越优先选择。

第十一章 布局和定位

布局表格
布局元素元素 值
float 浮动 left；right
clear left；right
position absolute
display
vertical-align
  1. 块元素从上到下，外边距会折叠，间距为较大的一个外边距；内联元素从左上到右下，会适应浏览器大小，外边距不会折叠。
  2. 布局设计：流体布局，随浏览器大小改变；冻结布局，锁定元素，在body增加一个id，并在css中制定其宽度；凝胶布局，在冻结基础上在设置外边距左右宽为auto。css表格布局，创建各种id，然后display属性，设置table，table-row，table-cell，表格边距只能在table元素下设置。
  3. 内联元素可以围绕float属性周围，用于分栏，不会感知absolute属性值，用于添加图标，不能再使用clear属性。
  4. position属性有4个值：static，absolute，fixed，relative
